## Builder Pattern

```java
public class GollekChatModel implements ChatLanguageModel {

    private final GollekClient client;
    private final String model;
    private final float temperature;

    private GollekChatModel(Builder builder) {
        this.client = builder.client;
        this.model = builder.model;
        this.temperature = builder.temperature;
    }

    public static Builder builder() {
        return new Builder();
    }

    public static class Builder {
        private GollekClient client;
        private String model;
        private float temperature = 0.7f;

        public Builder endpoint(String url) {
            this.client = new GollekClient(url);
            return this;
        }

        public Builder model(String model) {
            this.model = model;
            return this;
        }

        public Builder temperature(double t) {
            this.temperature = (float) t;
            return this;
        }

        public GollekChatModel build() {
            return new GollekChatModel(this);
        }
    }
}
```

# 🧠 4. Message Mapper (CRITICAL DESIGN)

LangChain4j → Gollek format

```java
public class GollekMessageMapper {

    public static String toPrompt(List<ChatMessage> messages) {
        StringBuilder sb = new StringBuilder();

        for (ChatMessage msg : messages) {
            switch (msg.type()) {
                case SYSTEM -> sb.append("[SYSTEM] ").append(msg.text()).append("\n");
                case USER -> sb.append("[USER] ").append(msg.text()).append("\n");
                case AI -> sb.append("[ASSISTANT] ").append(msg.text()).append("\n");
            }
        }

        return sb.toString();
    }
}
```

# ⚡ 5. Streaming Engine Bridge

```java
@Override
public void generate(List<ChatMessage> messages,
                     StreamingResponseHandler handler) {

    var request = GenerationRequest.builder()
        .prompt(GollekMessageMapper.toPrompt(messages))
        .stream(true)
        .build();

    client.generateStream(request)
        .onToken(handler::onNextToken)
        .onComplete(handler::onComplete)
        .onError(handler::onError);
}
```

## Adapter Layer

```java
public class GollekToolMapper {

    public static Optional<ToolExecutionRequest> map(String responseJson) {

        var node = parse(responseJson);

        if (!node.has("type") || !node.get("type").equals("tool_call")) {
            return Optional.empty();
        }

        return Optional.of(
            ToolExecutionRequest.builder()
                .name(node.get("name").asText())
                .arguments(node.get("arguments").toString())
                .build()
        );
    }
}
```

# 🧠 7. Embedding Model

```java
public class GollekEmbeddingModel implements EmbeddingModel {

    private final GollekClient client;

    @Override
    public Embedding embed(String text) {

        var res = client.embed(
            EmbeddingRequest.of(text)
        );

        return Embedding.from(res.getVector());
    }
}
```
