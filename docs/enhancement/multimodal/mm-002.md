# 🧠 4. Early Fusion Implementation (Concrete)

## Step-by-step

```id="d7d8x1"
[TEXT]  t1 t2 t3
[IMAGE] i1 i2 i3 i4

→ [t1 t2 t3 i1 i2 i3 i4]
```

## Code

```java
public class EarlyFusionEngine implements FusionEngine {

    @Override
    public FusionResult fuse(List<EncodedInput> inputs, FusionContext ctx) {

        List<float[]> fused = new ArrayList<>();
        List<Integer> positions = new ArrayList<>();

        int pos = 0;

        for (EncodedInput input : inputs) {
            for (int i = 0; i < input.tokenCount; i++) {
                fused.add(input.embeddings[i]);
                positions.add(pos++);
            }
        }

        return new FusionResult(
            toMatrix(fused),
            positions.stream().mapToInt(i -> i).toArray(),
            AttentionMask.full(fused.size())
        );
    }
}
```

## Code

```java
public class AttentionMask {

    boolean[][] mask;

    public static AttentionMask multimodal(int total, int textEnd) {

        boolean[][] m = new boolean[total][total];

        for (int i = 0; i < total; i++) {
            for (int j = 0; j < total; j++) {

                if (i < textEnd) {
                    m[i][j] = true; // text attends all
                } else {
                    m[i][j] = (j >= textEnd); // image attends image
                }
            }
        }

        return new AttentionMask(m);
    }
}
```
